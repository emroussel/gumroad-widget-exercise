# Gumroad Widget Clone

Features:

- Open Gumroad links in iframe popup
- Preload pages when hovering links
- Handle subdomains, but only when they're on the gum.co or gumroad.com domains
- Format links with Gumroad's brand if they have the `.gumroad-button` css class
- Handle `data-gumroad-embed-ignore` attribute to not open those links in iframe
